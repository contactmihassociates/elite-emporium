Orders are placed via WhatsApp

Customers fill the cart, then "Place Order via WhatsApp" sends a formatted message to +91 7358 650 774. The order is saved in the customer's browser history and printable as a GST invoice.

To centralize orders, integrate a payment gateway (see README §8) or wire the order JSON to Firestore via Firebase Functions.
